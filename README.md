# SamvaadAI

SamvaadAI is a sophisticated, AI-powered platform designed by **Yuvraj Singh Shekhawat**, a Data Scientist. It combines multi-modal AI capabilities into a single, intuitive interface, offering seamless interaction through text and image analysis.

## ✨ Key Features

- **🤖 Intelligent Chat**: Engaging, context-aware conversations powered by state-of-the-art language models.
- **🖼️ Image Analysis**: Extract insights, summarize visuals, identify objects, and create stories from your images.
- **🎨 Artisan (Image Generation)**: Transform text prompts into high-quality visuals using advanced diffusion models.
- **💎 Premium Design**: A sleek, modern UI featuring glassmorphism, smooth animations (Framer Motion), and a responsive layout.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **API**: Hugging Face Inference API
- **Styling**: Vanilla CSS with Tailwind utilities

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/[your-username]/samvaadai.git
   cd samvaadai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure API Key**:
   Replace the `API_KEY` in `src/lib/api.js` with your Hugging Face API key.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components/`: UI components (Chat, Analysis, Generation, Sidebar)
- `src/lib/api.js`: API integration logic
- `src/App.jsx`: Main application entry point
- `app.py`: Backend skeleton (prepared for future extensions)
